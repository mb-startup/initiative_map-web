1. Скачать openserver
2. Зайти в phpMyAdmin (Логин: root, без пароля)
3. Создать таблицу ini_bd в кодировке utf8_general_ci
4. Импортировать в тублицу базу данных из папки "База данных"
5. Закинуть весь репозиторий в папку domains в корневой папке программы openserver
6. Написать с строке браузера https://название папки в папке domains
7. Вы на сайте)
